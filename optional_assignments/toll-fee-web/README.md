## Application parts

- Vite
- React
- Typescript
- Material UI
- dayjs

## To run
- Run `yarn dev` to run development mode with HMR active
- Application available at http://localhost:3000

## Functionality
- Add vehicle type
- Add date and time
- Show passages
- Show total cost per day
